
# Mini Social Media App

A full-stack social media application where users can create posts, like, and comment — similar to a basic social feed.

---

## Features

* User Authentication (Signup & Login)
* Create Post (Text)
* View Public Feed
* Like / Unlike Posts
* Add Comments
* Show Username on Posts & Comments
* Search Posts
* Logout Functionality

---

##  Tech Stack

### Frontend:

* React.js
* CSS Modules

### Backend:

* Node.js
* Express.js

### Database:

* MongoDB (MongoDB Atlas)

### Other:

* JWT Authentication
* Axios

---

---


##  Environment Variables

Create a `.env` file in backend:

```
PORT=8000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```
